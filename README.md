## What is kebab-launcher?

kebab-launcher is a specialized, open-source custom Minecraft launcher developed using the Electron framework and Node.js. It is designed to provide a modern, customizable alternative to the default Minecraft launcher, allowing users to manage multiple profiles, modpacks, and game versions with ease.
